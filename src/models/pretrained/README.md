**Pretrained models**

This folder contains pretrained models for various languages. The naming scheme for the models is as follows:

- `{lang_code}_{model_type}_pretrained.pth`
    - `lang_code`: Language code (e.g., `fo` for Faroese)
    - `model_type`: Type of model (e.g., `baseline` for a basic model)
    - `pretrained`: Indicates that the model is pretrained

Use these models to perform Named Entity Recognition (NER) tasks in different languages.
